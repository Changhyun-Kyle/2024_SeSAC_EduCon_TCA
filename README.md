# 📖 목차
### - SwiftUI + TCA?
### - MVVM에 대한 고찰(feat. SwiftUI)
### - TCA(The Composable Architecture)
### - 기본 개념
  - **앱의 상태: State**
  - **상태 변화를 일으키기 위한 모든 동작: Action**
  - **변경을 처리한다: Reducer**
  - **의존성: Dependency**
  - **Action의 결과물: Effect**
